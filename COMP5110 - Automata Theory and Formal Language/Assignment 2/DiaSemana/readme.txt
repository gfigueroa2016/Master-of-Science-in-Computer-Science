DiaSemana GUI Form Description
================================================================================

This projects contains a visual form that when inserted a number from the range
of 1 to 7 it should return the day of the week pertaining to the number inserted.
if the user enters another number other than the range from 1 to 7, except for the number 9,
it should throw a warning to try again. If the user inserts 9, it should close the program.
